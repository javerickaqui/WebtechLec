What is the WWW?

  • It is the system of internet servers per website that is existing in the Internet now.
  • It was invented by Tim Berners-Lee in the year 1989 while he was working for Center for European Nuclear Research(CERN)
  • In 1994, Tim Berners-Lee founded the World Wide Web Consortium (W3C)
  • It was soon jointly developed by W3C and Internet Engineering Task Force(IETF)

What is HTTP?

  • It stands for Hyper Text Transfer Protocol.
  • It is like "blackboard" of the internet used to access or retrieve different resources that is published in the World Wide Web.

HTTP provides support for other functionalities just like:
  1. Cache Control
  2. Content Media Type
  3. Content/ Transfer
  4. Content Negotiation
  5. Client-Server Protocol Negotiations
  6. Persistent
  7. Request Pipelining
  8. Authentication/ Authorization
